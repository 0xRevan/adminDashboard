adminBoard
Project Overview
This project is a comprehensive admin dashboard designed to manage and display information in a user-friendly manner. It includes functionality for searching, viewing cryptocurrency prices, managing content, and connecting to a MetaMask wallet.

Features
Dynamic User Interface: Includes a responsive 6x6 card layout for managing various data points or functions.
Cryptocurrency Price Tracking: Displays real-time prices for Bitcoin (BTC), Ethereum (ETH), and PenDex (a custom token).
MetaMask Integration: Allows users to connect their MetaMask wallet directly from the dashboard.
Search Functionality: Users can search through the dashboard content with an integrated search bar.
